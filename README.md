This repository contains the files related to the SEPOC 2025 Industrial Challenge with Typhoon HIL.
1) File to design the type II Controller,
2) Simulation file in TyphoonSym,
3) Link to the presentation of the work entitled “CPT-Based Multifunctional Grid-Tied Inverter with Type II Controller”.

The team is affiliated with the Federal University of Santa Catarina (UFSC) and the São Paulo State University (UNESP). The Team Lead is Prof. Dr. Tiago Davi Curi Busarello, supported by the following Team Members:
